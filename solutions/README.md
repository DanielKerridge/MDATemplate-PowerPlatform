## How to Export

1. Go to [Power Apps Maker Portal](https://make.powerapps.com/)
2. Select your environment
3. Go to **Solutions**
4. Select **MDA Template**
5. Click **Export solution** > **Next** > **Unmanaged** > **Export**
6. Download the zip and place it here

## How to Import

1. Go to the target environment in [Power Apps Maker Portal](https://make.powerapps.com/)
2. Go to **Solutions** > **Import solution**
3. Select the `MDATemplate_unmanaged.zip` file
4. Follow the import wizard
5. After import, publish all customizations
